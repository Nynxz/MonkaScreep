## Install
[Screeps Doc: Advanced Grunt Usage ](https://docs.screeps.com/contributed/advanced_grunt.html)

1. `npm install` to get dependencies

2. You need your own `.screeps.json` in the root

```
  {
    "email": "USERNAME/EMAIL",
    "password": PASSWORD",
    "branch": "default",
    "ptr": false
  }
```
